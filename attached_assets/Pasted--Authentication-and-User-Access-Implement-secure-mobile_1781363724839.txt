### Authentication and User Access

Implement secure mobile number authentication.

#### OTP Login Flow

* User must enter their mobile number.
* Send a 6-digit OTP.
* Access to the application is allowed only after successful OTP verification.
* If the OTP is incorrect, expired, or invalid, deny access and show "Invalid OTP. Please try again."
* Limit OTP attempts to prevent abuse.
* Allow OTP resend after 30 seconds.
* Maintain secure user sessions.

### Trading Performance Heatmap (LeetCode Style)

Create a contribution-style heatmap similar to LeetCode submissions, but for trading performance.

#### Daily Heatmap Colors

##### Profit Days

* Dark Green = High Profit
* Medium Green = Moderate Profit
* Light Green = Small Profit

##### Loss Days

* Light Red = Small Loss
* Medium Red = Moderate Loss
* Dark Red = Heavy Loss

##### No Trade Days

* White box

  "No Trades"

##### Market Holidays

* Orange box
* Tooltip should display:
  "Market Holiday"

##### Weekends

* Grey box
* Tooltip should display:
  "Weekend"
